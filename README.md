EDDiscovery - What is it?

You play Elite Dangerous and have heard of EDDiscovery - and ended up here.  So what _is_ it?

In short it's a 3rd party tool that can track your Elite Dangerous travels, combat, trading, rank etc.

EDD works on the PC.  You can install it on your Elite PC, or on another laptop and use a share to obtain journal information from your Elite PC.  For Console commanders, you can run it on a laptop and configure it to download journal information from the Frontier servers

EDD also works (with limitations) on Linux using Mono 

EDDiscovery gives you the following major features:

* Shows your ED History
* Shows your current Materials State, Commodity State, Mission Lists
* Plan your Expeditions
* Has a 3D map of the galaxy showing travel history and data from Spansh or EDSM to aid your journey through the Elite Dangerous galaxy.
* A simple 2D map
* Roccat Style grid/Web browser support - use your tablet to display status and control the ship using a Roccat type grid.

EDDiscovery supports add-ons which give you the ability to:

* Vocalise Elite Events as they are received 
* Voice recognition - Control EDD using your voice using the Free Voice Control pack
* Keyboard shortcuts to control EDD
* Read and vocalise joystick inputs
* Place information on OBS
* Play MP3s when an event occurs using the soundblaster pack
* Get commands from the Elite local Chat interface using the Chat Console

EDDiscovery supports extending its operation using 3rd party DLL's

EDDiscovery supports Python plugins

EDDiscovery has been Translated into a few other languages.  You can change the language you want to see on the Settings Panel

EDDiscovery integrates with our partner Elite 3rd party developer tools:

* EDSM
* Inara
* Spansh
* IGAU
* Coriolis
* ED Shipyard
* ED Astro
* EDDN Network : Run by https://edcd.github.io/, this distributes information on shipyards, markets, stations, scans to the EDCD community (EDSM, EDDB, etc).
* Cannon community.
